# Cloud

## Definition

Within the HAIN framework, cloud refers to the centralized coordination layer of intelligent systems.

It includes shared infrastructure, large-scale orchestration, integration, and system-wide access to common intelligence resources.

## Role in the framework

Cloud remains essential because many AI-native systems still depend on centralized scale, shared model access, and broader coordination logic.

In HAIN, cloud is not treated as the only environment, but it remains one of the core layers of the system landscape.

## In one line

**Cloud is the centralized coordination and scale layer within the HAIN system landscape.**
