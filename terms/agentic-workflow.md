# Agentic Workflow

## Definition

An agentic workflow is a multi-step process in which intelligent software coordinates tools, memory, context, and actions in order to move toward a task outcome.

## Role in HAIN

Agentic workflows are one of the reasons the Hybrid AI Native framing is necessary.

They represent a shift from direct prompt-response interaction toward more structured, tool-using, state-aware system behavior. This makes AI systems more process-oriented and more layered.

## Why it matters

A system that supports agentic workflows is not only generating outputs. It is coordinating steps, maintaining context, and interacting with multiple resources across time and environment.

This introduces a different level of system design complexity.

## In one line

**An agentic workflow is a coordinated, multi-step process in which intelligent software uses tools, context, and actions to complete work.**
